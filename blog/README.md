## API 만들기

### 엔티티 구성하기

| 컬럼명     | 자료형          | null 허용 | 키   | 설명     |
|---------|--------------|---------|-----|--------|
| id      | BIGINT       | N       | 기본키 | 일련번호   |
| title   | VARCHAR(255) | N       |     | 게시글 제목 |
| content | VARCHAR(255) | N       |     | 게시글 내용 |

### 글 작성하기

`[POST] /posts`

제목, 내용을 입력하면 글을 작성할 수 있다.
